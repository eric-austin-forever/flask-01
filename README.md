## 修改密码
```
1. 添加点击跳转的链接
2. 添加视图函数，渲染指定的模板文件
3. 创建表单类，实例化一个表单对象，并把对象渲染的模板上
4. 校验原密码是否正确，将新密码更新到数据库中
```

## 找回密码
```
1. 在登录界面添加一个找回密码的链接
2. 渲染模板，用户提交用户名或者邮箱
3. 校验之后，向邮箱发送一封找回密码的邮件，包含该用户的信息
4. 添加重设密码的视图函数，用于处理用户找回密码的链接的请求
5. 更新新的密码存入数据库
```

## 修改邮箱
```
1. 添加点击跳转的链接
2. 渲染模板，给出输入新邮箱的输入框
3. 校验后（判断邮箱是否存在是否符合邮箱的要求）向新邮箱发送一封邮件，包含用户的信息
4. 处理点击邮件链接，添加一个方法，用于更新数据库中的邮箱
```

## 上传头像
```
1. 添加上传头像的链接
2. 渲染模板
3. 准备上传表单类
4. flask-uploads
5. 生成随机文件名，生成缩略图
6. 显示在用户名旁
```
